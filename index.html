Tinge

Free Screenshot Mac OS X Software

  • Download ZIP File

Inspiration.

We developed another screenshot capture application in order to get the best features the others have and git rid off the annoying behavior we could not live with.

User Experience

Tinge is Mac OS X application with the straightforward and standard controls. Just click Command+2, select the area and make simple marks on it. No the notorious "suns", disappearing windows, hidden menus and so on. Despite of that we avoided clattering the Dock with the extra icon - it appears only while an editor window is open and hides while Tinge sits in the backgrouind.

Dropbox as a cloud storage

There's definitely no need to create extra accounts nowdays to make the screenshots available online - there are lot of free cloud file storages all around. We have chosen Dropbox just because we used it for our needs but we are planning to add more alternatives.
